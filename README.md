# Instructions
## for backend
 1. npm init -y
 2. npm install express cors axios --save
 3. npm i nodemon --save
 4. npm install
 5. npm start

## for frontend
1. npm install vite@latest
2. npm install axios
3. npm run dev -- for running the application
